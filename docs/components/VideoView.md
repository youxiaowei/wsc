视频显示视图，配置参数如下：

变量 | 类型 | required | 描述
----|------|----------|-----
src | string | true | 来源
type | string | true | 视频格式类型
autoplay | boolean | false | 是否自动播放，默认为false
controls | boolean | false | 是否显示播放控件，默认为true
height | number | true | 视频控件的高度
width | number | false | 视频空间的宽度，默认为100%
