# SearchView
#author 李文奇
搜索输入框
className: SearchView
## 配置
变量 | 类型 | required | 描述
----|------|----------|-----
autoTrigger | bool | false | 未定议
filterKey | string | false | 未定议
placeholder | string | false | 输入框无输入时提示信息
例子：
```js
{
  type: 'SearchView',
  options: {
    autoTrigger:false,
    filterKey:"",
    placeholder:"请输入搜索内容"
  }
}
```
